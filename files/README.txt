FILES FOLDER
============

Place the following files in this folder:

1. schedule.pdf  — The studio class schedule.
                   The bot will send this file when a client taps "Розклад".
                   Replace whenever the schedule changes.

2. rules.pdf     — Studio rules document.
                   The bot will send this file when a client taps "Правила студії".

Supported formats: PDF, JPG, PNG, DOCX
If only one format is available, place that file and update config.json accordingly.
